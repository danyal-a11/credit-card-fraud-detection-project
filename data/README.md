# Data Directory

## Credit Card Fraud Detection Dataset

**Source:** [Kaggle - Credit Card Fraud Detection](https://www.kaggle.com/datasets/mlg-ulb/creditcardfraud)

**File:** `creditcard.csv` (144 MB)

### How to Get the Data:

1. Create a free Kaggle account
2. Download the dataset from the link above
3. Place `creditcard.csv` in the `data/raw/` directory

### Dataset Information:

- **Size:** 284,807 transactions
- **Features:** 31 (Time, V1-V28, Amount, Class)
- **Class Distribution:** 
  - Legitimate: 284,315 (99.827%)
  - Fraudulent: 492 (0.173%)

### Note:
The actual CSV file is not included in this repository due to GitHub's 100MB file size limit. This is standard practice for data science projects.
